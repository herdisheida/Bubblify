# Bubblify

**Bubblify** is a small school project built with **Next.js**.
It is a simple shopping application that allows user to browse products, add them to the cart and go thorugh a checkout process.


## How to Run

1. Install dependencies and Start the developement server

|            | go to directory | install dependencies | run           |
| ---------- | --------------- | -------------------- | ------------- |
| run server | `cd server`     | `npm install`        | `npm start`   |
| run app    | `cd my-app`     | `npm install`        | `npm run dev` |

2. Open the app in your browser

```
http://localhost:3000/bubbles
```

## Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
